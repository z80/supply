Microcontroller built in functions list!

trigger() This function returns boolean result. It returns true if trigger socket is shorted to ground and false otherwise.

setRtc( ms ) Sets real time clock time to specified number of milliseconds.

rtc() Returns current time in milliseconds.

msleep( ms ) Stops script execution for provided number of milliseconds.

setSerialEn( en, baud ) Turns serial port on or off. If second argument is provided it represends desired baud rate.

serialSend( data[], cnt ) Sends serial data through serial port. Data array and number of bytes to send should be provided.

serialReceive( data[], maxCnt ) Receives data through serial port. Data array and maximum bytes number should be provided.

setI2cSlaveEn( en, addr ) Turns I2C slave on or off. Optional second argument is desired I2C address.

setI2cEn( en ) Turns on oar off I2C master device.

i2cIo( addr, dataSend[], cntSend, dataReceive[], cntReceive, timeoutMs ) Data exchange through I2C master. Destination address should be provided as addr, dataSend - array of data to be sent, cntSend - bytes number to sent, dataReseive - array to receive data to, cntReceive - maximum number of bytes to receive, timeoutMs - timeout in milliseconds.

setWakeup( en ) Set wakeup pin value to either logical HIGH or LOW. To wakeup addon boards from hybernation transfer from LOW to HIGH should happen.

busbWrite( stri[] ) Write text string to USB.

io( at ) Returns IO array value at specifued with function argument position.

setIo( at, val ) Sets IO array at specified position to value provided.

setLed( mask ) Turns LEDs on or off depending on mask provided. Bit 0 corresponds to the first LED, bit 1 - to the second one.

setMoto( mask ) Turns motor controller channels on or off according to mask provided similarly to setLed( mask) command.

convSetBuck( mv ) Set buck voltage converter output voltage value in millivolts.

convSetBoost( mv ) Set boost voltage converter output value in millivolts.

convSetMinBoostVolt( mv ) Set minimum valur of boost voltage converter input. If input goes below this limit boost convertion decreases output.

convSetMaxBoostFill( percentOverTen ) Set PWM maximum fill for boost converter in %/10. By default it is 70%, e.i. 700. For regular applications there is no need in changing this parameter. Do it only if you do know what you do it for.

convSetBoostCurr( ma ) Set input curtent limit for boost converter in millivolts. If current consumption is close to this limit converter decreases its output voltage value.

convSetBuckCurr( ma ) Set input current limit for buck converter in millivolts. If input current value is close to this value converter decreases output voltage.

convSetBuckGain( percentOverTen ) Set buck converter gain value in percents over 10. Default value for buck coverter is 10%, e.i. 100. If load current isn't very high output voltage could be not very precise. In this case it might be useful to set smaller gain value.

convSetBoostGain( percentOverTen ) Set boost converter gain value in percents over 10. It is similar to convSetBuckGain( percentOverTen ). If load isn't very big voltage set precission could be not very big. And in order to increase output voltage precission it might be useful to set smaller gain value.

adcTemp() Returns temperature sensor output in Celsius over 10.

adcBatteryVolt() Returns input voltage value in millivolts.

adcBuckVolt() Returns measured buck voltage converter output in millivolts. It might be useful to compare this value to setpoint set with convSetBuck( mv ).

adcSolarVolt() Returns boost converter input voltage in millivolts.

adcBuckCurr() Returs buck voltage converter input current in millivolts.

adcBoostCurr() Returns boost voltage converter input current in millivolts.

addSetLed( index, arg ) Turns on or off LEDs on board index according to mask provided. Two leds correspond to 0 and 1 bits in the mask.

addSetMoto( index, arg ) Turns on or off motor drivers on board index according to arg provided. 8 motor outputs are controlled by 0...7 bits of arg.

addSetPwmPeriod( index, us ) Sets Pulse Width Modulation period on board index to be equal to us microseconds.

addSetPwm( index, mask, percentOver100 ) Sets PWM active duration pulse on board index at outputs chosen by the mask. Active phase is set using %/100 units

addInput( index ) Returns digital inputs state of board index as a bit mask.

addAdc( index, arg ) Returns ADC value on board index at arg analog input.
